【软件说明】
 * 软件名称：   编码器电机智能小车控制例程源码
				包含麦轮、四轮差速、两轮差速、阿克曼、三轮全向等底盘类型
				
 * 主要功能：   基本的正逆运动学解析
                PS2手柄控制小车运动
			   上位机软件控制小车运动

【硬件说明】
 * 硬件平台：   OpenCTR H60 V3.2
 * 处理器：     STM32F407VET6
 * 高速晶振：   8MHz
 * 总线时钟：   系统时钟 = SYCCLK = AHB1 = 168MHz
 * 硬件接口：   参考开源硬件原理图
   
   
 【协议说明】  
 * 串口协议:    塔克通用的X-Protocol协议
